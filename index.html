How to get your Firebase config:
1. Go to console.firebase.google.com
2. Click + Add project → name it "flowtrack" → Continue
3. In the project, click ⚙️ Project Settings → scroll to Your apps
4. Click </> Web → register app → copy the firebaseConfig object
5. Also go to Build → Firestore Database → Create database → Start in test mode
